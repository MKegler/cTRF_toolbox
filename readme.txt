cTRF_toolbox v0.1

Mikolaj Kegler

Imperial College London, 20.05.2018

Python implementation of the complex TRF modelling (cTRF) toolbox developed and applied in [Etard, et. al., 2018].

Attached files are:

cTRF.py - set of custom functions for cTRF modelling
demo.ipynb - example usage of the presented function

Required packages to run this demo:
MNE https://martinos.org/mne/dev/index.html
NumPy http://www.numpy.org/
SciPy https://www.scipy.org/
Matplotlib https://matplotlib.org/

(All the packages should install automatically with the recent version of MNE)

Example datasets that can be processed in the attached demo are avaliable under the following link: https://imperiallondon-my.sharepoint.com/:f:/g/personal/mak616_ic_ac_uk/En-V7CKltGdPqYKzd5xJ6a0BljqepRaHF-m9ta__LdtYhA?e=NgMXUv

For more details and commentary please see the jupyter notebook with the demo.

For any comments/bug reports/questions, please send feel free to contact me at mikolaj.kegler16@imperial.ac.uk.

Enjoy!

Reference: Etard, O., Kegler, M., Braiman, C., Forte A.E., Reichenbach, T. (2018) 'Real-time decoding of selective attention from the human auditory brainstem response to continuous speech', doi: https://doi.org/10.1101/259853
